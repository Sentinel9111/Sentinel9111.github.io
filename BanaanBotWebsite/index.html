Set up

There are 2 ways to set up the bot, these methods are for when you want the bot to only work in some channels but not all, if you want the bot to work in every channel you can just add the bot to the server with no further setup.

1:

  1. Add the Mee6 bot to your server and set it up so that everyone gets a custom member role.
  2. Remove permissions for @everyone to view channels.
  3. Add permissions to your custom member role to view channels.
  4. Add BanaanBot to your Discord Server.
  5. Give BanaanBot special permissions in the channels that you want it to work to view the channels.

Issues with this method:

  • This could cause problems with other bots in your Discord server.
  • You have to add Mee6 and set that bot up correctly in your Discord server.

2:

  1. Add BanaanBot to your Discord server.
  2. Remove permissions to view channels for every channel that you don't want it to work in.

Issues with this method:

  • You have to remove permissions in every channel that you don't want it to work in.
  • You have to remove the permissions again for new channels.

Known bugs

  • The bot changes all letters of your message to lowercase.
  • The bot corrects the already corrected words.
